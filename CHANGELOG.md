## [1.0.0] - 2026-06-22

### Added

- Initial release.
- Clean Architecture project structure.
- CQRS with MediatR.
- ASP.NET Core Web API.
- Entity Framework Core.
- SQL Server support.
- ASP.NET Core Identity.
- JWT authentication.
- Refresh token support.
- FluentValidation.
- AutoMapper.
- Serilog logging.
- Global exception middleware.
- Repository pattern.
- Unit tests project.
- Integration tests project.
- Sample project with WorkItems feature.
- Empty template.
